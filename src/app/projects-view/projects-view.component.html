Chatter

A chat based application made in Flutter using dart and Firebase. It has OTP verification for users, allows users to send text, images, gifs and videos along with ability to post statuses.
It has more features like replying to messages as well as showing online/offline based on the app state.

  • Flutter
  • Firebase
  • Dart
  • Riverpod
